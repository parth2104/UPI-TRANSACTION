# UPI Transactions Analysis Dashboard (Power BI)

## Project Overview
This project presents an interactive UPI Transactions Dashboard built using Power BI, focusing on transaction patterns across different currencies, merchants, cities, and age groups.

The dashboard helps analyze financial activity, understand spending behavior by age group, and identify merchant and currency trends using dynamic filters.

---

##  Key Features

- Multi-Currency Analysis (INR, USD, EUR, GBP)  
- Merchant-wise Transaction Insights  
- Age Group-Based Analysis  
- City-wise Breakdown (Bangalore, Delhi, Hyderabad, Mumbai)  
- Payment Method Analysis  
- Device Type Tracking  
- Bank-wise Transactions (Sent & Received)  
- Monthly Transaction Trends (2024)  
- Toggle Between Line & Column Charts  
- Interactive Filters (Slicers)  

---

## Dashboard Highlights

### Currency & City Transaction Table
- Displays transaction amounts and remaining balances  
- Segmented by city and currency  
- Helps compare financial activity across regions  

### Merchant & Age Group Insights
- Analyze how different age groups interact with merchants  
- Identify spending patterns across categories  

### Monthly Transaction Trends
- Visualizes transaction patterns across months  
- Highlights peak and low-performing periods
